> 图片代理服务，用于绕过某些网站的图片防盗链

## Usage

```bash
go run main.go
```

`http://localhost:8888/?url=<图片URL>`

示例：

`http://localhost:8888/?url=https://i1.hdslb.com/bfs/archive/2a4374c3aab4e7c53e9f78c3dbdacfa3474f76b8.jpg`
